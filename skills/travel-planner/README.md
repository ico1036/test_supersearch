# 🗺️ Travel Planner Skill

테마 기반 여행 일정 작성 스킬 - 검증된 추천과 다양한 옵션 제공

## 사용법

```
/travel-planner
```

또는 자연어로:
```
"성수동에서 데이트 코스 짜줘"
"제주도 3박4일 힐링 여행 계획해줘"
"30명 팀빌딩 저녁 식사 장소 추천"
```

---

## 핵심 기능

### 1. 테마 기반 코스 설계
- 힐링, 맛집 투어, 역사 탐방, 데이트, 팀빌딩 등

### 2. 4단계 검증 시스템
모든 추천 장소에 대해:
- ✅ 영업 상태 확인
- ✅ 위치 적절성 검토
- ✅ 운영 시간 확인
- ✅ 수용 인원 체크

### 3. 5개 이상 대안 옵션
각 시간대별 메인 추천 + 5개 이상의 백업 옵션

### 4. 대규모 그룹 모드 (10명+)
- 거리/골목 상권 기반 분산 활동
- 서브그룹 분배 계획
- 집결 포인트 명시

### 5. 신뢰도 있는 소스
- 네이버 플레이스, 카카오맵 (공식)
- 망고플레이트, 트립어드바이저 (리뷰)
- 최신 블로그/인스타 (트렌드)

---

## 예시

### 소규모 (2-4명)
```
입력: "홍대에서 2명 카페 데이트 코스, 토요일 오후"

출력:
- 최적 동선의 카페 3-4곳 추천
- 각 카페별 5개 대안
- 영업시간, 위치, 분위기 검증
- 예약 필요 여부
```

### 대규모 (10명+)
```
입력: "성수동 20명 팀점심, 화요일 12시"

출력:
- 성수동 핵심 거리 선정
- 거리 내 분산 식사 옵션 5개
- 팀별 배정 제안
- 집결 장소 및 시간
```
